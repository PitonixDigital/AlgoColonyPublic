# CryptoDCA

A Python tool for performing **Dollar-Cost Averaging (DCA)** calculations on cryptocurrencies using Binance historical data.

## Features
- Supports **multiple symbols** at once
- Fetches **monthly historical data** from Binance
- Calculates **portfolio value, total tokens, ROI**
- Generates a **growth curve chart** with a baseline of total invested
- Command-line interface for quick usage

## Requirements
- Python 3.9+
- pandas
- matplotlib
- python-binance

Install dependencies:
```bash
pip install pandas matplotlib python-binance
